
# 2023_W_Team6_Jonah - ProjectTracker

# Summary
The main goal of this app is to provide an easy to use interface for
tracking the status of projects by breaking them up into smaller tasks and
tracking these tasks. Users can also collaborate with other users on
projects by sharing them.

# Tech Stack
* ASP.NET Core MVC + React SPA
* Entity Framework Core
* SQL Server
